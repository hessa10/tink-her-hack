# KnowYourKappal

A compatibility test application for Valentine's Day!

## How to Run

1.  Navigate to the `tink her hack` folder on your desktop.
2.  Double-click `index.html` to open it in your web browser.
3.  Enjoy finding your relationship personality!

## Features

-   **Basic Info**: Collects user details and interests.
-   **Compatibility Quiz**: 10 questions to analyze your relationship style.
-   **Results**:
    -   Compatibility Score (0-100%)
    -   Personality Type (Communicator, Independent, etc.)
    -   Green & Red Flags Analysis
    -   Curated Date Ideas based on your interests.

## Tech Stack

-   HTML5
-   CSS3 (Premium styling with gradients and animations)
-   JavaScript (Logic for scoring and dynamic content)
